## Getting Started with Beamer

### Basic Presentation Structure

<CodeGroup>
```latex basic-presentation.tex
\documentclass{beamer}

% Theme selection
\usetheme{Madrid}
\usecolortheme{default}

% Packages
\usepackage[utf8]{inputenc}
\usepackage{graphicx}

% Presentation info
\title{Your Presentation Title}
\subtitle{Optional Subtitle}
\author{Your Name}
\institute{Your Institution}
\date{\today}

\begin{document}

% Title slide
\frame{\titlepage}

% Table of contents
\begin{frame}
\frametitle{Outline}
\tableofcontents
\end{frame}

% Section 1
\section{Introduction}
\begin{frame}
\frametitle{Introduction}
\begin{itemize}
  \item First point
  \item Second point
  \item Third point
\end{itemize}
\end{frame}

% Section 2
\section{Main Content}
\begin{frame}
\frametitle{Key Concepts}
Your content here...
\end{frame}

% Thank you slide
\begin{frame}
\centering
\Huge Thank You!
\end{frame}

\end{document}
```
</CodeGroup>

### Frame Basics

<CodeGroup>
```latex frame-basics.tex
% Simple frame
\begin{frame}
\frametitle{Slide Title}
Content goes here
\end{frame}

% Frame with subtitle
\begin{frame}
\frametitle{Main Title}
\framesubtitle{Subtitle}
Content with subtitle
\end{frame}

% Frame options
\begin{frame}[fragile] % For verbatim content
\frametitle{Code Example}
\begin{verbatim}
def hello():
    print("Hello, World!")
\end{verbatim}
\end{frame}

% Plain frame (no decorations)
\begin{frame}[plain]
\centering
\Huge Special Slide
\end{frame}

% Centered content
\begin{frame}[c] % t=top, c=center, b=bottom
\frametitle{Vertically Centered}
This content is centered vertically
\end{frame}
```
</CodeGroup>

### Custom Colors

<CodeGroup>
```latex custom-colors.tex
\documentclass{beamer}
\usepackage{xcolor}

% Define custom colors
\definecolor{myblue}{RGB}{0,114,189}
\definecolor{myred}{RGB}{217,83,79}
\definecolor{mygreen}{RGB}{92,184,92}

% Apply custom colors
\setbeamercolor{structure}{fg=myblue}
\setbeamercolor{alerted text}{fg=myred}
\setbeamercolor{example text}{fg=mygreen}

% Custom color for specific elements
\setbeamercolor{frametitle}{bg=myblue!20,fg=myblue}
\setbeamercolor{block title}{bg=myblue,fg=white}
\setbeamercolor{block body}{bg=myblue!10}

\begin{document}

\begin{frame}
\frametitle{Custom Colored Slide}
Normal text with \alert{alerted text} and \structure{structured text}.
\end{frame}

\end{document}
```
</CodeGroup>

## Overlays and Animation

### Basic Overlays

<CodeGroup>
```latex overlays-basic.tex
\begin{frame}
\frametitle{Step-by-Step Reveal}

\begin{itemize}
  \item<1-> First item appears on slide 1
  \item<2-> Second item appears on slide 2
  \item<3-> Third item appears on slide 3
  \item<4-> Fourth item appears on slide 4
\end{itemize}

\only<1>{Text only on slide 1}
\only<2>{Text only on slide 2}
\only<3->{Text from slide 3 onwards}

\visible<2->{This is visible from slide 2}
\uncover<3->{This is uncovered from slide 3}
\end{frame}
```
</CodeGroup>

### Advanced Overlays

<CodeGroup>
```latex overlays-advanced.tex
\begin{frame}
\frametitle{Complex Animations}

% Highlighting text
This is \alert<2>{important} and this is \alert<3>{also important}.

% Changing content
\only<1>{Initial equation: $E = mc$}
\only<2>{Oops, forgot something: $E = mc^2$}

% Incremental equations
\begin{align}
  f(x) &= x^2 + 2x + 1 \\
  \onslide<2->&= (x+1)^2 \\
  \onslide<3->&> 0 \text{ for all } x \neq -1
\end{align}

% Dynamic blocks
\begin{block}<4->{Conclusion}
The function has a minimum at $x = -1$.
\end{block}
\end{frame}
```
</CodeGroup>

## Math in Presentations

### Mathematical Content

<CodeGroup>
```latex math-beamer.tex
\begin{frame}
\frametitle{Mathematical Theorems}

\begin{theorem}[Fundamental Theorem of Calculus]
If $f$ is continuous on $[a,b]$, then
\[
\int_a^b f(x)\,dx = F(b) - F(a)
\]
where $F'(x) = f(x)$.
\end{theorem}

\pause

\begin{proof}
\begin{enumerate}
  \item<2-> Define $G(x) = \int_a^x f(t)\,dt$
  \item<3-> Show that $G'(x) = f(x)$
  \item<4-> Apply the Mean Value Theorem
\end{enumerate}
\visible<5->{Therefore, the theorem is proved. \qed}
\end{proof}

\end{frame}
```
</CodeGroup>

% Custom title page
\setbeamertemplate{title page}{
  \vbox{}
  \vfill
  \begin{centering}
    {\usebeamercolor[fg]{title}\usebeamerfont{title}\inserttitle\par}

    \vskip2em

    \vskip1em
    {\usebeamerfont{institute}\insertinstitute\par}

## Best Practices

<Tip>
**Presentation design tips:**

1. **Simplicity**: Less is more - avoid cluttered slides
2. **Consistency**: Use the same theme throughout
3. **Font size**: Ensure text is readable from the back of the room
4. **Colors**: Use high contrast for visibility
5. **Animations**: Use sparingly and purposefully
6. **Time**: Plan ~2-3 minutes per slide
7. **Backup**: Always have a PDF version ready
</Tip>

## Common Issues

<Warning>
**Troubleshooting Beamer:**

1. **Compilation errors**: Use `[fragile]` for verbatim content
2. **Missing themes**: Ensure full LaTeX distribution is installed
3. **Overlay issues**: Check overlay specifications syntax
4. **Font problems**: Some themes require specific fonts
5. **PDF size**: Optimize images before including them
</Warning>

## Quick Reference

### Frame Options
```latex
[plain]            % No header/footer
[fragile]          % For verbatim
[allowframebreaks] % Auto-split long content
[shrink]           % Fit content
[squeeze]          % Reduce spacing
[t]                % Top alignment
[c]                % Center alignment
[b]                % Bottom alignment
```

### Overlay Specifications
```latex
<1>    % Only on slide 1
<1->   % From slide 1 onwards
<-3>   % Until slide 3
<2-4>  % From slide 2 to 4
<1,3>  % On slides 1 and 3
```
