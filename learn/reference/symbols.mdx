## Matrix and Array Environments

### Matrix Types

<CodeGroup>
```latex parentheses-matrix.tex
% Parentheses matrix
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
```

```latex bracket-matrix.tex
% Bracket matrix
\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}
```

```latex determinant.tex
% Determinant
\begin{vmatrix}
a & b \\
c & d
\end{vmatrix}
```

```latex norm.tex
% Norm
\begin{Vmatrix}
a & b \\
c & d
\end{Vmatrix}
```
</CodeGroup>

## Special Packages for More Symbols

<Info>
Some symbols require special packages:

```latex
\usepackage{amssymb}    % Extended symbols
\usepackage{amsfonts}   % Math fonts
\usepackage{latexsym}   % LaTeX symbols
\usepackage{textcomp}   % Text symbols
\usepackage{wasysym}    % Wasy symbols
\usepackage{marvosym}   % Martin Vogel's symbols
```
</Info>

## Symbol Search Tips

<Tip>
**Can't find a symbol?**
1. Try [Detexify](http://detexify.kirelabs.org/) - draw the symbol to find its LaTeX command
2. Use the [Comprehensive LaTeX Symbol List](http://tug.ctan.org/info/symbols/comprehensive/symbols-a4.pdf)
3. Search for "latex [description] symbol" online
4. Check if you need a special package
</Tip>

<Warning>
Remember that some symbols require math mode (`$...$` or `\[...\]`) while others work in text mode. When in doubt, use math mode.
</Warning>
