This reference covers the most important LaTeX packages organized by functionality. Each entry includes the package purpose, basic usage, and practical examples.

<Info>
**Package installation**: LaTeX Cloud Studio includes all standard packages. No installation needed - just use `\usepackage{packagename}`.
</Info>

## Essential Packages

### Core Document Setup

<CodeGroup>
```latex essential-packages.tex
% Input encoding (for older documents)
\usepackage[utf8]{inputenc}

% Font encoding
\usepackage[T1]{fontenc}

% Language support
\usepackage[english]{babel}
% or multiple languages
\usepackage[english,spanish,french]{babel}

% Better fonts
\usepackage{lmodern}

% Microtype (better typography)
\usepackage{microtype}
```
</CodeGroup>

### Page Layout

#### geometry
Controls page dimensions and margins.

<CodeGroup>
```latex geometry-package.tex
\usepackage{geometry}

% Common configurations
\geometry{margin=1in}
\geometry{left=1.5in, right=1in, top=1in, bottom=1in}
\geometry{a4paper, margin=2.5cm}
\geometry{letterpaper, landscape}

% Advanced options
\geometry{
  paperwidth=210mm,
  paperheight=297mm,
  left=25mm,
  right=25mm,
  top=30mm,
  bottom=30mm,
  headheight=15pt,
  headsep=10mm,
  footskip=15mm
}
```
</CodeGroup>

#### fancyhdr
Custom headers and footers.

<CodeGroup>
```latex fancyhdr-package.tex
\usepackage{fancyhdr}
\pagestyle{fancy}

% Clear defaults
\fancyhf{}

% Custom headers/footers
\fancyhead[L]{Left Header}
\fancyhead[C]{Center Header}
\fancyhead[R]{\thepage}
\fancyfoot[C]{Footer Text}

% Different odd/even pages
\fancyhead[LE,RO]{\thepage}
\fancyhead[LO,RE]{\leftmark}

% Custom styles
\fancypagestyle{plain}{
  \fancyhf{}
  \fancyfoot[C]{\thepage}
}
```
</CodeGroup>

## Mathematics Packages

### amsmath
Extended math environments and commands.

<CodeGroup>
```latex amsmath-package.tex
\usepackage{amsmath}

% Provides environments:
\begin{align}
  f(x) &= x^2 + 2x + 1 \\
  &= (x + 1)^2
\end{align}

\begin{gather}
  a = b + c \\
  d = e + f
\end{gather}

\begin{split}
  (a + b)^3 &= a^3 + 3a^2b \\
  &\quad + 3ab^2 + b^3
\end{split}

% Commands
\text{text in math}
\tfrac{a}{b}  % Text-style fraction
\dfrac{a}{b}  % Display-style fraction
\binom{n}{k}  % Binomial coefficient
```
</CodeGroup>

### amssymb
Additional mathematical symbols.

<CodeGroup>
```latex amssymb-package.tex
\usepackage{amssymb}

% Provides symbols like:
\mathbb{R}    % Real numbers
\mathbb{Z}    % Integers
\mathbb{N}    % Natural numbers
\varnothing   % Empty set
\therefore    % Therefore
\because      % Because
\blacksquare  % QED symbol
\leqslant     % Less or equal (slanted)
\geqslant     % Greater or equal (slanted)
```
</CodeGroup>

### amsthm
Theorem environments.

<CodeGroup>
```latex amsthm-package.tex
\usepackage{amsthm}

% Define theorem environments
\newtheorem{theorem}{Theorem}[section]
\newtheorem{lemma}[theorem]{Lemma}
\newtheorem{corollary}[theorem]{Corollary}

\theoremstyle{definition}
\newtheorem{definition}[theorem]{Definition}

\theoremstyle{remark}
\newtheorem{remark}[theorem]{Remark}

% Usage
\begin{theorem}[Fermat's Last Theorem]
No three positive integers $a$, $b$, and $c$ satisfy...
\end{theorem}

\begin{proof}
The proof is left as an exercise.
\end{proof}
```
</CodeGroup>

### mathtools
Enhanced version of amsmath.

<CodeGroup>
```latex mathtools-package.tex
\usepackage{mathtools}

% Additional features
\begin{align}
  f(x) &= \begin{cases}
    x^2 & \text{if } x > 0 \\
    0 & \text{otherwise}
  \end{cases}
\end{align}

% Better spacing
\coloneqq     % :=
\Coloneqq     % ::=
\eqqcolon     % =:

% Paired delimiters
\DeclarePairedDelimiter\abs{\lvert}{\rvert}
\DeclarePairedDelimiter\norm{\lVert}{\rVert}

$\abs{x}$, $\abs*{\frac{a}{b}}$
$\norm{v}$, $\norm*{\sum_{i=1}^n x_i}$
```
</CodeGroup>

## Graphics Packages

### graphicx
Including and manipulating images.

<CodeGroup>
```latex graphicx-package.tex
\usepackage{graphicx}

% Include images
\includegraphics{image.png}
\includegraphics[width=5cm]{image.png}
\includegraphics[height=3cm]{image.png}
\includegraphics[scale=0.5]{image.png}
\includegraphics[angle=90]{image.png}

% Set graphics path
\graphicspath{{images/}{figures/}{../pics/}}

% Figure environment
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.8\textwidth]{diagram}
  \caption{Figure caption}
  \label{fig:diagram}
\end{figure}
```
</CodeGroup>

### tikz
Creating graphics programmatically.

<CodeGroup>
```latex tikz-package.tex
\usepackage{tikz}
\usetikzlibrary{arrows,shapes,positioning}

% Simple drawing
\begin{tikzpicture}
  \draw (0,0) -- (2,0) -- (1,1.732) -- cycle;
  \draw[red,thick] (1,0.577) circle (0.577);
  \node at (1,0.577) {Center};
\end{tikzpicture}

% Flowchart
\begin{tikzpicture}[
  box/.style={rectangle, draw, minimum width=2cm, minimum height=1cm},
  arrow/.style={->, thick}
]
  \node[box] (A) {Start};
  \node[box, below=of A] (B) {Process};
  \node[box, below=of B] (C) {End};
  
  \draw[arrow] (A) -- (B);
  \draw[arrow] (B) -- (C);
\end{tikzpicture}
```
</CodeGroup>

<CodeGroup>

\newcolumntype{L}[1]{>{\raggedright\arraybackslash}p{#1}}
\newcolumntype{R}[1]{>{\raggedleft\arraybackslash}p{#1}}

## Citation Packages

### biblatex
Modern bibliography management.

<CodeGroup>
```latex biblatex-package.tex
\usepackage[
  backend=biber,
  style=authoryear,  % or numeric, alphabetic, etc.
  sorting=nyt,
  maxbibnames=10,
  maxcitenames=2
]{biblatex}

\addbibresource{references.bib}

% In document
\textcite{smith2020}  % Smith (2020)
\parencite{smith2020} % (Smith 2020)
\footcite{smith2020}  % Footnote citation

% Multiple citations
\parencite{ref1,ref2,ref3}

% Print bibliography
\printbibliography
\printbibliography[type=book,title={Books}]
```
</CodeGroup>

### natbib
Traditional bibliography management.

<CodeGroup>
```latex natbib-package.tex
\usepackage[round]{natbib}

% Citation commands
\cite{key}          % Jones et al. (1990)

\citet{key}         % Jones et al. (1990)
\citealp{key}       % Jones et al., 1990
\citeauthor{key}    % Jones et al.

% Multiple citations
\citep{key1,key2}   % (Jones et al., 1990; Smith, 1991)

% Bibliography
\bibliographystyle{plainnat}
\bibliography{references}
```
</CodeGroup>

## Code Listing Packages

### listings
Source code formatting.

<CodeGroup>
```latex listings-package.tex
\usepackage{listings}
\usepackage{xcolor}

\lstset{
  language=Python,
  basicstyle=\ttfamily\small,
  keywordstyle=\color{blue}\bfseries,
  stringstyle=\color{red},
  commentstyle=\color{gray}\itshape,
  numbers=left,
  numberstyle=\tiny\color{gray},
  stepnumber=1,
  numbersep=5pt,
  backgroundcolor=\color{gray!10},
  frame=single,
  breaklines=true,
  captionpos=b
}

\begin{lstlisting}[caption=Python example]
def factorial(n):
    """Calculate factorial of n"""
    if n <= 1:
        return 1
    return n * factorial(n - 1)
\end{lstlisting}

% Inline code
\lstinline|print("Hello")|
```
</CodeGroup>

### minted (requires --shell-escape)
Syntax highlighting with Pygments.

<CodeGroup>
```latex minted-package.tex
\usepackage{minted}

\begin{minted}[
  frame=lines,
  framesep=2mm,
  baselinestretch=1.2,
  bgcolor=gray!10,
  fontsize=\footnotesize,
  linenos
]{python}
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[0]
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]
    return quicksort(left) + [pivot] + quicksort(right)
\end{minted}

% Inline
\mintinline{python}{lambda x: x**2}
```
</CodeGroup>

### algorithm2e
Algorithm typesetting.

<CodeGroup>
```latex algorithm2e-package.tex
\usepackage{algorithm2e}

\begin{algorithm}[H]
\SetAlgoLined
\KwData{Array $A$ of $n$ elements}
\KwResult{Sorted array $A$}
\For{$i \leftarrow 1$ \KwTo $n-1$}{
  $key \leftarrow A[i]$\;
  $j \leftarrow i - 1$\;
  \While{$j \geq 0$ \textbf{and} $A[j] > key$}{
    $A[j+1] \leftarrow A[j]$\;
    $j \leftarrow j - 1$\;
  }
  $A[j+1] \leftarrow key$\;
}
\caption{Insertion Sort}
\end{algorithm}
```
</CodeGroup>

## Best Practices

<Tip>
**Package usage tips:**

1. **Load order matters**: Some packages must be loaded in specific order
2. **Avoid conflicts**: Some packages are incompatible with others
3. **Options**: Many packages accept options to customize behavior
4. **Documentation**: Use `texdoc packagename` to read documentation
5. **Minimal loading**: Only load packages you actually use
6. **Updates**: Keep packages updated for bug fixes and features
</Tip>

## Common Package Conflicts

<Warning>
**Known conflicts and solutions:**

- `hyperref`: Load last (with few exceptions like `cleveref`)
- `float` + `floatrow`: Use one or the other
- `subfigure` vs `subcaption`: Use `subcaption` (newer)
- `times` + `mathptmx`: Use `newtxtext` + `newtxmath` instead
- `enumerate` + `enumitem`: Use `enumitem` only
</Warning>

## Package Loading Order

Recommended loading order for common packages:

```latex
\documentclass{article}

% 1. Font encoding
\usepackage[T1]{fontenc}
\usepackage[utf8]{inputenc}

% 2. Language
\usepackage[english]{babel}

% 3. Fonts
\usepackage{lmodern}

% 4. Page layout
\usepackage{geometry}

% 5. Math packages
\usepackage{amsmath,amssymb,amsthm}

% 6. Graphics
\usepackage{graphicx}
\usepackage{xcolor}

% 7. Tables
\usepackage{booktabs}
\usepackage{array}

% 8. Lists
\usepackage{enumitem}

% 9. Bibliography
\usepackage{biblatex}

% 10. Hyperref (near last)
\usepackage{hyperref}

% 11. Cleveref (after hyperref)
\usepackage{cleveref}
```
